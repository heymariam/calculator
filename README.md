# Age Calculator

A simple and interactive age calculator web application built with vanilla JavaScript, HTML, and CSS.

## Features

- Calculate exact age in years, months, and days from a birth date
- User-friendly date input with validation
- Real-time calculation
- Responsive design

## How to Use

1. Open `index.html` in your web browser
2. Select your birth date using the date picker
3. Click the "Calculate" button
4. Your exact age will be displayed

## Technologies Used

- **HTML5** - Structure
- **CSS3** - Styling and responsive design
- **JavaScript (Vanilla)** - Age calculation logic

## File Structure

- `index.html` - Main HTML file
- `index.css` - Styling
- `index.js` - JavaScript logic

## Demo

Simply open the `index.html` file in any modern web browser to see it in action.
